markdown学习记录

Markdown 段落

Markdown 段落没有特殊的格式，段落的换行是使用两个以上空格加上回车

Markdown 标题

使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号

在 Markdown 中插入一张图片并指定其大小

img src=”路径/到/你的/图片.jpg” alt=”图片描述” width=”300” height=”200”
